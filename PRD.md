# PRD：朋友圈照片与截图分析 H5

## 1. 背景与目标
- 用户上传 5 张及以上对方朋友圈照片或截图，调用豆包大模型进行分析
- 目标：帮助用户更好地与对方建立友好关系，成为好朋友

## 2. 需求核对（用户输入理解）
- 产品形态：H5 网页
- 输入：用户上传对方朋友圈照片或截图，数量不少于 5 张
- 处理：调用豆包大模型，模型为 doubao-seed-1.6-thinking
- 输出：生成对该人的性格特征分析，并给出交友建议
- 调用文档：火山方舟 API（Responses API）

## 3. 用户画像与场景
- 用户画像：希望与对方更好相处的普通用户
- 典型场景：添加对方好友后，想通过朋友圈照片与截图了解兴趣与性格

## 4. 核心用户流程
1. 进入 H5 上传页面
2. 阅读说明与隐私提示并确认
3. 上传照片或截图（≥5 张），预览与删除不满意图片
4. 点击“开始分析”
5. 等待分析完成
6. 查看分析结果与交友建议
7. 可复制建议话术或保存结果

## 5. 功能需求
### 5.0 页面结构
- 仅一个页面：上传页面
- 页面内包含上传区、分析按钮、结果展示区
### 5.1 图片上传
- 支持多图选择与拖拽上传
- 必须满足 5 张及以上
- 单张图片大小限制（建议 10MB）
- 支持上传后预览与删除
- 支持 JPG/PNG

### 5.2 分析请求
- 触发条件：图片数量 ≥ 5
- 调用模型：doubao-seed-1.6-thinking
- 提示词包含：
  - 用户目标：以结交朋友为目的
  - 角色与语气：友好、尊重、现实可行
  - 输出结构化建议（见 5.3）
  - system prompt：你是一个寻找朋友的客户，对于个人的心理和外在表现，有非常强的洞察，也有一套很厉害的识别朋友匹配度的技巧！擅长于输出简短但有效的分析和建议。

### 5.3 输出内容结构
- 性格特征：外向/内向倾向、社交偏好、情绪风格
- 画像总结：兴趣、生活方式、价值观倾向
- 互动策略：聊天切入点、话题建议
- 风险提示：可能的误读与谨慎建议
- 行动清单：3-5 条可执行步骤
- 推荐话术：2-3 条个性化开场白

### 5.4 结果展示
- 分段展示，支持一键复制
- 结果可重新生成
- 显示分析免责声明

## 6. 非功能需求
- 性能：上传后 30s 内给出分析结果
- 可用性：移动端优先适配
- 可靠性：失败提示明确且可重试
- 安全：不在前端暴露 API Key
- 隐私：仅用于分析，支持一键删除

## 7. 交互与文案要点
- 上传区提示：“请上传 5 张及以上照片或截图”
- 分析按钮：满足数量后可点击
- 结果页提示：“分析基于公开内容，结果仅供参考”

## 8. 数据与合规
- 用户确认已获得对方内容使用授权
- 不保存或在分析后立即删除图片
- 不向第三方分享数据

## 9. API 约束与接入
- Base URL：https://ark.cn-beijing.volces.com/api/v3
- 模型：doubao-seed-1.6-thinking
- API Key：ARK_API_KEY 环境变量
- 建议使用 Responses API（兼容 OpenAI SDK）

## 10. 关键指标
- 上传成功率
- 分析完成率
- 结果复制率
- 用户满意度

## 11. 里程碑
- MVP：上传 + 分析 + 结果展示
- V1：话术模板与二次生成
